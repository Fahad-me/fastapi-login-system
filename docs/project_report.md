# Abstract

The FastAPI Authentication Platform is a secure web application developed using FastAPI and PostgreSQL.

The purpose of this project is to understand and implement a real-world authentication system.

The application provides secure user registration and login using JWT authentication while storing user information inside a PostgreSQL database.

The frontend is developed using HTML, CSS, JavaScript, and Jinja2 templates to provide a responsive and modern user interface.

The project follows a modular architecture by separating routes, services, schemas, models, templates, and static resources.

This architecture improves scalability, maintainability, and code readability.

# Chapter 1

# Introduction

Authentication is one of the most essential components of modern web applications.

Every secure system requires a mechanism to identify users, verify their credentials, and provide controlled access to protected resources.

This project demonstrates how authentication systems are built using modern backend technologies.

Instead of relying on traditional session-based authentication, this project implements JSON Web Token (JWT) authentication for secure communication.

The project also demonstrates password hashing using bcrypt, secure database management using PostgreSQL, and clean project architecture using FastAPI.

# Chapter 2

# Problem Statement

Many beginner authentication systems are developed inside a single Python file without following any software architecture.

These projects become difficult to maintain and extend.

Problems include:

- Poor code organization

- Hardcoded credentials

- Plain text passwords

- No authentication token

- No database normalization

- No scalability
